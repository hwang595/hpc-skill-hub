# Archive Review Checklist

- Confirm the source path is the intended dataset, not a broad project or home directory.
- Remove or exclude temporary files, caches, logs, failed-run outputs, and private data.
- Confirm required metadata, README files, licenses, and citation notes are included.
- Confirm the archive path is on storage suitable for large sequential writes.
- Generate checksums for the final archive before transfer, publication, or handoff.
