# Code Of Conduct

HPC Skill Hub is intended to be useful across institutions, disciplines, and
experience levels. Contributors are expected to be respectful, patient, and
specific when giving feedback.

## Expected Behavior

- Assume good intent while still reviewing carefully.
- Explain cluster-specific assumptions clearly.
- Give actionable feedback instead of vague criticism.
- Respect institutional security and privacy boundaries.
- Help newer HPC users learn safe habits.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Publishing secrets, private hostnames, or user data.
- Encouraging unsafe use of shared systems.
- Dismissing valid operational or security concerns.

Maintainers may edit, hide, or reject contributions that violate these norms.
